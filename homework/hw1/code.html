Sorting Algorithms

Overview

This page contains an explanation of each algorithm to help understand the underlying function of each of the sorting algorithms.
Specific code for various programming languages are provided via a linked website.

Bubble Sort

Bubble sort works by walking through each element of a given input list, for as many times as there are elements.
It starts from the leftmost element and works its way to the rightmost unsorted element, comparing two elements at a time.
If the left element is greater than the right element, then the two elements get swapped, putting the greater value on the right.
This code checks to see if any values were changed during a loop, and if not it breaks out of the sort.

Quicksort

Quicksort (Quick Sort) works with a method called "divide and conquer" in which the algorithm picks an element as a pivot - typically the middle or last element - and the list is split into values smaller than the pivot and values larger than the pivot.
After splitting the lists, the process gets repeated with each of the sides, splitting up each sides into smaller sides and sorting them.
Eventually, the lists get reconstructed back together in their new sorted forms.

Mergesort

Mergesort (Merge Sort) follows the "divide and conquer" method of sorting where the main list is split in half until each list is only a single element.
Once the values have been broken down to a one-item list, they are merged back together, two lists at a time, until the full list is reassembled and sorted.

Insertion Sort

Insertion Sort works by walking through an array of elements and inserting each element in its sorted spot.
It differs from Bubble Sort as each element is only visited once in one for-loop and uses a while-loop to move up the array until it's in its sorted position.

Selection Sort

Selection Sort is one of the simplest sorting algorithms. It works by searching through a list for the smallest value.
Once it finds the smallest value, it inserts it to the beginning of the list. Then, it repeats the loop, starting from the next element, and searches again for the next smallest element.
Once the next smallest value is found, it gets inserted right in front of the previous smallest value.
This cycle repeats until the entire list is sorted.
